# blog
TheBreeze的博客

## 关于

这是一个基于VuePress的博客项目，用于记录生活、分享技术和记录成长。

## 技术栈

- **框架**: VuePress 2.x
- **主题**: VuePress Theme Hope
- **语言**: TypeScript
- **部署**: GitHub Pages / Vercel

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
my-docs/
├── src/
│   ├── demo/          # 演示页面
│   ├── posts/         # 博客文章
│   ├── zh/            # 中文内容
│   └── intro.md       # 介绍页面
├── package.json
└── README.md
```
