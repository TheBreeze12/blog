# Vue

## 前端框架

### Vue

Vue 是一个用于构建用户界面的 JavaScript 框架。它专注于视图层，采用组件化开发，易于上手，性能出色。Vue 的核心库只关注视图层，易于与第三方库或既有项目整合。

### React

React 是一个用于构建用户界面的 JavaScript 库。它专注于视图层，采用组件化开发，易于上手，性能出色。React 的核心库只关注视图层，易于与第三方库或既有项目整合。
